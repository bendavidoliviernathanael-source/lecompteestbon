# lecompteestbon
Le compte est bon est un jeu qui consiste à chercher une séquence de calculs qui permet d'obtenir un résultat qui est égal ou le plus proche possible d'un résultat attendu en utilisant des nombres d'une collection de nombres fournie et des opérations parmi +, -, x et /.
Ce dépôt introduit un programme qui cherche une séquence de calculs.


Le jeu ne permet pas de réutiliser plusieurs fois un même nombre de la collection de nombres fournie initialement.
Le jeu permet d'utiliser les nombres obtenus avec des calculs.
Une réalisation d'une séquence de calculs conduit à utiliser des nombres de la collection de nombres fournie initialement et à produire des nombres utilisables pour d'autres calculs.
