# lecompteestbon
Le compte est bon est un jeu qui consiste à chercher une séquence de calculs qui permet d'obtenir un résultat qui est égal ou le plus proche possible d'un résultat attendu en utilisant des nombres d'une collection de nombres fournie et des opérations parmi +, -, x et /.
Ce dépôt introduit un programme qui cherche une séquence de calculs.
